# 🎯 AI 3 Saudara - Application Suite

Welcome to AI 3 Saudara! A collection of modern web applications powered by Gemini AI and advanced task management.

## 📚 Available Applications

### 🎨 [Gemini Studio](https://rickyprayudha0000-maker.github.io/ai-3-saudara/index.html)
**AI-Powered Image & Video Generator**

Generate stunning images and videos using Google's Gemini AI with:
- 🖼️ Text-to-Image Generation with multiple art styles
- 🎬 Text-to-Video with seamless looping
- 🔄 Frame-based video transitions (Cross Fade, Motion Blur, Morph, Optical Flow)
- 📅 Up to 10x video loops with smooth transitions
- 🔑 Secure API Key Management

**[Open Gemini Studio →](https://rickyprayudha0000-maker.github.io/ai-3-saudara/index.html)**

---

### ✅ [TaskFlow](https://rickyprayudha0000-maker.github.io/ai-3-saudara/todo.html)
**Advanced Todo List with Local Storage**

Manage your tasks efficiently with:
- ✏️ Create, edit, and delete tasks
- 🔴 Priority Levels (High, Medium, Low)
- 📁 Categories (Work, Personal, Shopping, Health, Other)
- 📅 Due date tracking
- 🔍 Filter by status, priority, and category
- 📊 Real-time statistics (completion rate, active tasks, etc.)
- 💾 All data saved to Local Storage
- ⌨️ Keyboard shortcuts (Ctrl+N to add, Esc to close)

**[Open TaskFlow →](https://rickyprayudha0000-maker.github.io/ai-3-saudara/todo.html)**
